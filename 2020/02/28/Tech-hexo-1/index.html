Hexo(1)-Hello Hexo

关于搭建博客

事情的起由是大概一年前买了个域名，然后正好把它解析到我买的服务器上，搭建了一个基于WordPress的动态博客，主要是在LAMP上做的。当时也就搭着玩的没想写东西，然而在某个重要的时间点后，服务器IP被墙了…

之后就考虑搭一个静态的不被墙的博客，主要看了Jekyll和Hexo，前者是用Ruby的而且Github支持比较好，后者用的Node.js而且用户也很多。但是我看中了Hexo快速、简洁且高效的特性(实际上是看上某些themes了…)，所以索性上船了，配合Github Pages还是很方便的。

主题用的是Freemind.386，喜欢这种风格的界面。其实我更想做成类似BOOTSTRA.386，可是我是前端菜鸡，上一次用还是5年以前简单的HTML+CSS，甚至没用过JavaScript。哪天等我变强了，我再改改吧。
